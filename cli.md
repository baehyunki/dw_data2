## 경로를 확인하는 방법

```bash
pwd
```

## 현재 경로에서 탐색기를 띄우는 방법

```bash
exploror .
```

## 하위 경로로 이동하는명령어
cd = change directory

```bash
cd 폴더명
```

## 현재 폴더안에 있는 내용을 확인할때
 
```bash
 ls
```

## 현재 폴더에서 디렉토리를 만들때

```bash
mkdir 만들고싶은_폴더명
```

## 파일을 생성하는 명령

여러개를 만들때는 띄어쓰기로 구분

```bash
touch index.html style.css
```
